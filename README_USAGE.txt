Usage Guide — TheorySmith Pro

1. Input a Theory

Type any physics or hypothetical concept into the UI.

Examples: - “A particle that accelerates exponentially in curved
spacetime.” - “A universe where gravity repels.”

2. System Outputs

-   Full theoretical breakdown
-   Mathematical modeling
-   Graphs & simulations
-   Plausibility evaluation

3. Visual Output

Includes: - Equations - Simulation plots - Time‑evolution dynamics -
Explanations

4. Export

You can export: - PNG graphs - Theory text - Full reports
